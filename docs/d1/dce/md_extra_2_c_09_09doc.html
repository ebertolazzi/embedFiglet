C++ usage

This library implement a subset of Figlet capability, i.e. print large letters out of ordinary text

_ _ _ _ _ _
| (_) | _____ | |_| |__ (_)___
| | | |/ / _ \ | __| '_ \| / __|
| | | < __/ | |_| | | | \__ \_
|_|_|_|\_\___| \__|_| |_|_|___(_)

Differently of standard FIGlet command (http://www.figlet.org/) this print can be done inside a user program using run time generated string as follows

#include "Figlet.hh"
#include <sstream>
using namespace std;
int
main() {
Figlet::standard.print("Fractions");
for ( int i = 2; i <= 4; ++i ) {
ostringstream ss;
ss << "5/" << i << " = " << 5.0/i;
Figlet::small.print(ss.str().c_str());
}
cout << "ALL DONE!\n";
return 0;
}

A simple ruby script permits to convert figlet .flf files to structures which can be embedded in the library.

Available printing mode

Only four mode of print the same string:

Printing mode

  • FIGLET_SMUSHED
  • standard way of figlet print
  • FIGLET_PACKED
  • letters are moved left but no overlappingare permitted
  • FIGLET_FULLWIDTH
  • letters are printed at width defined in thefont
  • FIGLET_MONOSPACED
  • letters are printed with equal width

Available framing mode

Only two framing are available:

Framing mode

  • FIGLET_SINGLE
  • single frame around a string
  • FIGLET_DOUBLE
  • double frame around a string

The effect are the following

Figlet::small.printFramed("SINGLE",cout,Figlet::FIGLET_SINGLE);

Compile and test

Edit makefile file to match compiler of your OS and do:

make

To run the test

make run

or use rakefile

# on linux os
rake build_linux
# on mac
rake build_osx
# on windows
rake build_win
